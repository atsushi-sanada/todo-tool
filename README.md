# TODOアプリ

シンプルで使いやすいTODOアプリケーションです。

## 機能

- タスクの追加・編集・削除
- タスクの完了状態管理
- ローカルストレージによるデータ永続化
- レスポンシブデザイン対応

## 使い方

1. ブラウザで `index.html` を開く
2. タイトルを入力してタスクを追加
3. チェックボックスでタスクの完了状態を変更
4. 編集ボタンでタスクを編集
5. 削除ボタンでタスクを削除

## テスト方法

### 手動テスト
1. ブラウザで `index.html` を開く
2. 各機能をテストする
   - タスクの追加（タイトル必須、説明任意）
   - タスクの一覧表示
   - タスクの完了状態変更
   - タスクの編集
   - タスクの削除
   - データの永続化（ページを再読み込みして確認）
3. エラーケースをテストする
   - タイトルを空にして追加
   - 長すぎるタイトル・説明を入力
4. レスポンシブデザインをテストする
   - ブラウザのウィンドウサイズを変更
   - 開発者ツールでデバイスモードを使用

### テスト計画書
詳細なテスト計画は `.cursor_workflow/test-plan.md` を参照してください。

## 技術スタック

- Node.js: 不要（Vanilla JSのため）
