# TODOアプリ

シンプルで使いやすいTODOアプリケーションです。

## 機能

- タスクの追加・編集・削除
- タスクの完了状態管理
- ローカルストレージによるデータ永続化
- レスポンシブデザイン対応

## 使い方

1. ブラウザで `index.html` を開く
2. タイトルを入力してタスクを追加
3. チェックボックスでタスクの完了状態を変更
4. 編集ボタンでタスクを編集
5. 削除ボタンでタスクを削除

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- LocalStorage API

## 開発環境

- Node.js: 不要（Vanilla JSのため）
- ブラウザ: モダンブラウザ（Chrome、Firefox、Edge、Safari）
