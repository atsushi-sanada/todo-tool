# Diagram Board

## 概要

Cacooライクな図表編集を、ブラウザだけで実行できるWebアプリです。  
ノード（四角形・円・ノート）の作成、ドラッグ移動、ノード同士の接続を行えます。  
データはローカル保存でき、JSON入出力とPNGエクスポートにも対応しています。

### このツールについて

- 図形の追加、移動、接続、削除
- プロパティ編集（ラベル・座標・サイズ・色）
- LocalStorage保存、JSON読込/出力、PNG出力

---

## クイックスタート

1. `index.html` をブラウザで開く
2. 左パネルでツール（四角形/円/ノート）を選び、キャンバスをクリックする
3. 上部の「保存」または「JSON出力」でデータを保存する

---

## セットアップ

- 前提条件: モダンブラウザ（Chrome / Edge / Firefox / Safari）

1. このリポジトリをローカルに配置する
2. `index.html` をダブルクリック、またはブラウザで開く
3. 必要に応じてブラウザのローカルストレージを有効にする

---

## 使い方

1. 左パネルで「接続」を選び、接続元ノード→接続先ノードの順にクリックする
2. 右パネルで選択中ノードのラベル・位置・サイズ・色を編集し、「適用」を押す
3. 上部の「読込」でJSONを取り込み、「PNG出力」で画像として保存する

---

## その他

- ショートカット: `Ctrl/Cmd + Z`（元に戻す）、`Ctrl/Cmd + Y`（やり直し）
- Space + ドラッグでキャンバス移動、ホイールでズーム
- 詳細はコード内ドキュメント参照
